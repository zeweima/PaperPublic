# Digest common rules

**Read this file at the start of any digest task.**

## Notes links

Include `· [notes](../notes/<id>.md)` whenever `papers/notes/<id>.md` exists for a paper, appended after the DOI link in any per-paper line:
- daily: Top picks, By theme, Quick scan
- weekly: Read this weekend
- monthly: Reading queue

Do **not** include notes links inside weekly or monthly synthesis prose paragraphs — those use inline DOI citations only.

## DOI links

Use DOI URLs for all paper links when available. arXiv URLs are fine for arXiv preprints. Never fabricate URLs.

## Theme assignment

Use the `Tags` line in each paper's note to assign theme. A paper appears under at most one theme heading per section — pick the dominant tag.

Standard themes (use these labels exactly):
- Hydrology & water resources
- Biogeochemistry & soils
- Land-surface & agroecosystem modeling
- Machine learning / methods
- Remote sensing & observations
- Geomorphology & geology
- Climate & global change

## Accuracy

Never fabricate numbers, authors, or findings. If a paper's note doesn't have a specific number, describe direction qualitatively or write "see paper".

## Tone

Dense but readable. Researcher-to-researcher, not press-release. Numbers > adjectives. No emojis. No "Key takeaways" subheadings inside paragraphs.

## Output

Print the output file path on the last line of your response.
