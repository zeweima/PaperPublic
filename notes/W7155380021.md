# A novel memory-based deep learning framework for reliable joint estimation of daily vegetation carbon fluxes

**Authors:** Ximing Chen, Jiejun Huang, Han Zhou et al.
**Venue:** Agricultural and Forest Meteorology · **Date:** 2026-04-24 · **DOI:** https://doi.org/10.1016/j.agrformet.2026.111198
**Score:** 9/10<sup> top pick</sup>
**Source:** full text

## TL;DR
DemLSTM — a dual-encoder multi-task LSTM — jointly estimates GPP, RECO, and NEE with higher accuracy and physical consistency than RF, XGBoost, and vanilla LSTM across 9 plant functional types.

## Summary
- **Problem:** Most data-driven carbon flux models estimate GPP, RECO, and NEE independently, ignore vegetation memory effects (lagged responses to climate), and exhibit poor physical consistency (large carbon closure bias |RECO−GPP−NEE|).
- **Method:** DemLSTM uses two independent LSTM encoders (one for meteorological drivers — Ta, VPD, LST, precipitation, soil moisture, SOC; one for vegetation indices — kNDVI, EVI, NDWI) feeding a fusion layer with three parallel output heads for simultaneous GPP/RECO/NEE prediction. Input sequence length is set per PFT via Granger causality tests (lag range 1–90 days). Trained on 447,218 site-days from FLUXNET2015 + AmeriFlux (213 sites, 9 PFTs); transferability tested on independent ICOS Warm Winter dataset (73 European sites). XAI (gradient-based feature importance) used to attribute lagged driver contributions.
- **Key result:** Site-based 5-fold CV: R² = 0.743 / 0.625 / 0.471 for GPP / RECO / NEE. Generalization (ICOS Warm Winter): R² = 0.698 / 0.594 / 0.497, RMSE = 2.203 / 1.721 / 1.758 g C m⁻² day⁻¹. Median carbon closure bias = 0.025 g C m⁻² day⁻¹ vs. 0.406 (RF), 0.431 (XGBoost), 0.406 (vanilla LSTM). Linear probes of DemLSTM internal states predict vegetation transpiration (median r = 0.89) and canopy conductance (r = 0.85), outperforming vanilla LSTM probes. XAI confirms antecedent shortwave radiation, VPD, and air temperature as dominant lagged GPP drivers; deep-rooted ecosystems (DBF, EBF, ENF, MF) show weaker soil moisture memory than shallow-rooted ones.
- **Why it matters:** Joint multi-task learning with explicit memory encoding substantially closes the carbon closure gap, improving physical realism without process-based constraints. The linear probe result suggests the model implicitly learns carbon–water coupling, offering a path to estimate evapotranspiration in data-sparse regions from carbon flux model internals.
- **Caveats:** ERA5 reanalysis meteorological inputs inherit reanalysis uncertainties; PML-V2 transpiration/conductance used as linear probe targets are themselves modelled, not observed. Performance on CRO and SH remains lower (R² ~0.52–0.53 for GPP). Transferability tested only within FLUXNET-style tower networks; regional upscaling not demonstrated.
