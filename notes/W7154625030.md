# [Interpretable feature incorporation machine-learning framework for flood magnitude estimation](https://doi.org/10.5194/hess-30-2135-2026)

**Authors:** Emma Ford, Manuela I. Brunner, Hannah Christensen et al.
**Venue:** Hydrology and Earth System Sciences · **Date:** 2026-04-16 · **DOI:** https://doi.org/10.5194/hess-30-2135-2026
**Score:** <unclear from abstract>/10
**Source:** full text

## TL;DR

Synoptic weather patterns co-occur with UK winter floods but add minimal predictive skill over catchment attributes and event-day precipitation alone.

## Summary

- **Problem:** Whether MO-30 synoptic weather patterns (WPs) improve large-sample ML flood magnitude estimation beyond local hydrometeorological and catchment predictors is unknown; prior UK models have not systematically tested atmospheric circulation as a feature set.
- **Method:** Random Forest regression trained on 134 near-natural UK benchmark catchments (1969–2021), predicting winter peak-over-threshold flood magnitudes (≥99th percentile specific discharge, mm d⁻¹); seven cumulative feature sets added sequentially — spatial coordinates, WP category, antecedent WPs (1–3 d prior), static catchment attributes, event-day hydromet variables, antecedent precipitation (1–3 d); SHAP used for interpretability; 63 model configurations across UK national + 8 regional samples; temporal train/test split (80/20).
- **Key result:** UK pooled model peaks at R² = 0.84 (feature set 6) and R² = 0.83 (pruned set 7); adding WP/antecedent-WP predictors (feature sets 2–3) does not significantly improve R² over the spatial baseline in most regions; largest R² gains occur when event-day hydromet and antecedent precipitation are included (sets 5–6). SHAP ranks aridity index as the top UK predictor, followed by event-day precipitation and antecedent precipitation indices. Regional R² ranges from 0.33 (CEE) to 0.83 (SW). UK pooled model outperforms regional models in 54.9% vs 45.1% of matched catchments.
- **Why it matters:** Establishes that synoptic-scale WPs are useful for identifying flood-conducive conditions but carry negligible incremental skill for peak magnitude prediction in a large-sample ML framework; justifies physically pruned, interpretable predictor sets and provides methodological guidance for integrating atmospheric predictors into flood estimation models.
- **Caveats:** Analysis restricted to winter (DJF) floods in near-natural UK catchments; CEE region (groundwater/chalk-dominated) poorly explained (R² = 0.33), suggesting framework struggles where slow hydrological processes dominate; models systematically underestimate upper-tail magnitudes (negative PBIAS across most regions); single 80/20 temporal split may not capture non-stationarity.
