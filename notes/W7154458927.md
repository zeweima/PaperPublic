# [Process-Oriented Calibration of a Turbulence Scheme in the DOE's Global Storm-Resolving Model Using Machine Learning](https://doi.org/10.1029/2025gl120241)

**Authors:** Y Zhang, Tianning Su, Hewei Tang et al.
**Venue:** Geophysical Research Letters · **Date:** 2026-04-15 · **DOI:** https://doi.org/10.1029/2025gl120241
**Score:** 9/10<sup> top pick</sup>
**Source:** full text

## TL;DR

ML surrogates + MCMC jointly calibrate 9 SHOC turbulence parameters across two ARM convective regimes, substantially improving boundary-layer clouds and radiative effects in DOE's SCREAM model.

## Summary

- **Problem:** Sub-grid turbulence parameterizations in global storm-resolving models carry large parametric uncertainty; traditional global tuning introduces compensating errors and is computationally prohibitive at kilometer scale.
- **Method:** 5,000-member perturbed-parameter ensembles in doubly periodic SCREAM (DP-SCREAM) for two ARM SGP regimes — clear-sky dry convective boundary layer (LAFE) and fair-weather shallow cumulus (CASS composite of 76 days). Fourier Neural Operator surrogates emulate vertical velocity variance w′² (MAPE < 2%); deep neural networks emulate cloud base height / cloud top height (R² > 0.93). Constrained MCMC (affine-invariant emcee sampler) minimizes a composite cost function against LES benchmarks. Top-20% posterior cluster medoids form the calibrated physics ensemble (CPE).
- **Key result:** CPE captures observed diurnal growth and decay of cloud fraction and its vertical distribution; reduces midday w′² bias relative to default. Cloud fraction improvement is emergent — CF was not an explicit calibration target but improved through joint constraint of CBH/CTH and w′². CPE shifts modeled PCF, LWP, cloud-layer thickness, and SW_CRE toward ARM observations versus the default parameter set. Sensitivity analysis identifies four dominant SHOC parameters: L_fac (mixing-length scale), C_kh (scalar eddy diffusivity), Lambda_low (lower-bound stability correction), and C_qw2 (moisture-variance tuning).
- **Why it matters:** Demonstrates process-oriented, multi-regime ML-assisted calibration as a computationally tractable alternative to ad hoc global tuning for high-resolution ESMs; shallow-cumulus layer depth in SCREAM can be substantially improved through parametric calibration without structural changes. Framework is repeatable and scalable to additional ARM case-library regimes.
- **Caveats:** Framework tested on only two land-based convective regimes (SGP site); cloud microphysics not jointly calibrated; global SCREAM evaluation is beyond scope — improvements in regional/global simulations not yet demonstrated. LWP shows residual positive bias partly attributable to microphysical processes not targeted by SHOC calibration.
